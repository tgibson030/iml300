Website Critique

Layout

1. Good Layout

This layout caught my eye with its use of color and a disrupted grid system. It still would have looked good if it was 4x4 blocks all the way down the page, but mixing up the proportions at the top is refreshing and highlights key information. The colored boxes all complement each other in a way that shows some personality without being distracting.
